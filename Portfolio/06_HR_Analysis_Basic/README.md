# HRAnalyticsPython

## İK Performans Analizi Projesi (HR Analytics Project)

Bu proje, 1000 kişilik sanal bir çalışan veri seti üzerinden İnsan Kaynakları analitiği yapmayı amaçlar.

### İçerik
1.  **Veri Üretimi (`01_veri_olusturma.py`):** Python (Faker, Numpy) kullanılarak gerçekçi bir İK verisi üretilir.
2.  **Veri Analizi (`02_veri_analizi.py`):** Pandas ve Seaborn kullanılarak maaş, performans ve tecrübe analizleri yapılır.

### Kullanılan Teknolojiler
*   Python
*   Pandas
*   Numpy
*   Seaborn
*   Matplotlib
*   Faker
