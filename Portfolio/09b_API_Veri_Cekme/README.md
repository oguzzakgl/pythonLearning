# 09b — API'den Veri Çekme (Ön Alıştırma)

Bu klasör, **10_Borsa_Analizi** projesine geçmeden önce
"API nedir, nasıl veri çekilir?" sorusunu Python ile adım adım cevaplar.

## İçerik

| Dosya | Konu |
|---|---|
| `01_api_nedir.py` | REST API kavramı, `requests` kütüphanesi |
| `02_public_api.py` | Ücretsiz & kayıt gerektirmeyen API'ler (Open-Meteo, JokeAPI) |
| `03_yfinance_temeller.py` | Yahoo Finance — hisse & kripto verisi çekme |
| `04_coklu_hisse.py` | Döngüyle birden fazla hisse, veriyi birleştirme |
| `05_kaydet_yukle.py` | CSV / JSON kaydetme, tekrar yükleme |

## Kurulum

```bash
pip install requests yfinance pandas
```
